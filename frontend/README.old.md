# Frontend - React

This folder will contain the React web application for:
- Doctor and patient dashboards
- Secure messaging and telehealth
- Patient monitoring and alerts
- Care coordination workflows

## Getting Started
1. Create the React app:
   ```bash
   npx create-react-app .
   ```
2. Install dependencies for healthcare and compliance:
   ```bash
   npm install axios react-router-dom @mui/material @emotion/react @emotion/styled
   ```
3. Start the development server:
   ```bash
   npm start
   ```

See `/docs/AI_Med_Agent_Publication_Strategy.md` for compliance and deployment details.
