# Mobile - React Native

This folder will contain React Native apps for:
- iOS (App Store)
- Android (Google Play)
- Patient and doctor mobile workflows
- Secure notifications and messaging

## Getting Started
1. Initialize the React Native app:
   ```bash
   npx react-native init MedAgentMobile
   ```
2. Install dependencies:
   ```bash
   npm install @react-navigation/native axios
   ```
3. Run on iOS:
   ```bash
   npx react-native run-ios
   ```
4. Run on Android:
   ```bash
   npx react-native run-android
   ```

See `/docs/AI_Med_Agent_Publication_Strategy.md` for compliance and publication details.
